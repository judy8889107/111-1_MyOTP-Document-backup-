使用最新人工智能深度學習技術——深度卷積神經網絡。它會將噪點和鋸齒的部分進行補充，實現圖片的無損放大。查看示例圖片

和其他的放大方式有什麽不同？

使用其他的放大方法如PS或PhotoZoom，放大的圖片後依然有明顯的模糊感，邊緣的重影以及噪點。我們產品的工作原理，是通過神經網絡，針對放大圖片的線條、顏色、網點等特點，做特殊的算法調整，所以放大效果非常出色, 色彩保留較好, 圖片邊緣也不會有毛刺和重影,。更重要的是，影響畫質的噪點基本在放大的圖片上看不出來。

什麽圖片放大效果最好？

動漫、插畫圖片的放大幾乎可以說是完美的。將小圖片放大後，無論是色彩、細節、邊緣，效果都很出色。同時也兼容普通的照片放大。

最大可以上傳多大圖片？

目前免費版3000x3000px, 5M; 付費版50M。

放大速度如何？

開始放大後會顯示預估的處理時間。依據不同的放大倍數和圖片尺寸，放大時間會有不同，從幾分鐘到幾十分鐘不等。壹般實際放大的時間比預估的時間要稍短。

為什麽會放大失敗？

受到網絡環境和服務器同時使用人數的影響，有很小幾率會出現放大失敗的情況。如果妳碰到這種情況，請過幾分鐘再試試。

開始放大以後需要壹直保持瀏覽器打開嗎？

如果妳沒有登錄，那麽需要壹直保持瀏覽器打開，否則將丟失放大的圖片。如果妳已經登錄了，那麽可以關閉瀏覽器，我們支持離線放大，過壹段時間再來下載就可以啦。

如何查看自己的放大歷史記錄？

為了維持這個工具的服務器開支，我們提供收費的放大服務。付費升級賬戶後妳可以使用獨立的高性能服務器，放大圖片更快、更多、更穩定。

我上傳的圖片會怎麼處理呢？

請放心，用戶上傳的圖片和放大後的圖片會在15天后自動刪除。圖片地址鏈接都是加密保存，除非用戶主動分享，否則其他人是無法查看到該圖片的。

放大圖片接口  

    curl -F 'conf={"style": "art", "noise": "3", "x2": "1", "input": "YOUR_IMAGE_URL"}' \
         -H 'X-API-KEY:' \
         https://bigjpg.com/api/task/

    import requests
    import json

    data = {
        'style': 'art',
        'noise': '3',
        'x2': '1',
        'input': 'YOUR_IMAGE_URL'
    }

    r = requests.post(
        url='https://bigjpg.com/api/task/',
        headers={'X-API-KEY': ''},
        data={'conf': json.dumps(data)}
    )
    print(r.json())



- conf 參數是JSON 格式
- style 參數有 'art', 'photo' 分別表示 '卡通插畫', '照片'
- noise 參數有 '-1', '0', '1 ', '2', '3' 分別表示降噪程度 '無', '低', '中', '>高', '最高'
- x2 參數有 '1', '2', ' 3', '4' 分別表示 2x, 4x, 8x, 16x
- input 參數是需要放大的圖片鏈接
